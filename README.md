Library Management API

Описание проекта
  Система управления библиотекой позволяет пользователям:
  Добавлять, просматривать, редактировать и удалять авторов и книги.
  Добавлять книги в список избранных.
  Фильтровать и искать книги по автору, жанру и дате публикации.
  Сортировать книги по дате публикации, имени автора и жанру.
  Получать уведомления о новых и юбилейных книгах через периодические задачи.
  Проект реализован с использованием Django, Django REST Framework, Celery и Redis, с аутентификацией через JWT.


🛠 Стек технологий
  Python 3.12+
  Django 4.x
  Django REST Framework
  djangorestframework-simplejwt (JWT аутентификация)
  PostgreSQL (или SQLite для локальной разработки)
  Celery + Redis (для периодических задач)
  Docker + Docker Compose (опционально для локального развёртывания)
  Swagger (документация API)

⚡ Основные функции
Пользователи
  Регистрация, вход и выход
  JWT аутентификация
Авторы
  Создание, просмотр, редактирование и удаление авторов
  Поля: first_name, last_name, biography, date_of_birth, date_of_death (необязательное)
Книги
  Создание, просмотр, редактирование и удаление книг
  Поля: title, summary, isbn, authors (связь с несколькими авторами), publication_date, genre
Избранные книги
  Добавление книг в список избранных
  Просмотр, удаление и очистка списка избранных
Фильтрация и поиск
  Фильтрация по авторам, жанрам, дате публикации
  Поиск по названию книги и фамилии автора
  Сортировка по дате публикации, имени автора и жанру
Периодические задачи
  Уведомления о книгах, добавленных за последние 24 часа
  Уведомления о юбилейных книгах (5, 10, 20 лет и т.д.)


API Документация
  Swagger UI доступен по адресу:

Установка и запуск проекта

  Клонировать репозиторий: 
    https://github.com/tidi18/telecom-.git

  Перейдите в корневую директорию:  
    cd library

  Запуск приложения Соберите и запустите контейнеры:
    docker-compose build
    docker-compose up -d


После запуска контейнеров вы можете получить доступ к приложению по адресу http://127.0.0.1:8000/
Административная панель Вы можете получить доступ к админ-панели по адресу http://127.0.0.1:8000/admin/ с использованием следующих учетных данных:
